# <a name="compare-differences-between-azure-offers-in-the-csp-program"></a>CSP 프로그램에서 Azure 제품 간의 차이점 비교

**적절한 역할**

- 관리 에이전트
- 청구 관리자
- 글로벌 관리자
- 기술 지원팀 상담원
- 영업 상담원
