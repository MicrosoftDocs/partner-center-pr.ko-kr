## <a name="updated-promotions-for-office-365"></a><a id="14"/></a>Office 365에 대해 업데이트된 프로모션

### <a name="categories"></a>범주

- 날짜: 2020-04-30
- 다음을 입력합니다. 기능



### <a name="summary"></a>요약

Microsoft는 2020년 5월 4일(오전 12:01 UTC)에 Office 365 E1 및 Microsoft 365 Business Basic에 대한 새로운 프로모션을 도입하여 신규 고객을 위한 Office 365 E1의 현재 6개월 평가판을 대체합니다.

### <a name="impacted-audience"></a>영향을 받는 대상

CSP(클라우드 솔루션 공급자) 프로그램을 통해 거래하는 모든 파트너 이 프로모션은 순 신규 상용 고객에게만 적용됩니다.

### <a name="details"></a>세부 정보

이 전례 없는 시기에 [Microsoft의 고객을 위한 노력](https://www.microsoft.com/microsoft-365/blog/2020/03/05/our-commitment-to-customers-during-covid-19/)의 일환으로, Office 365 E1의 6개월 평가판이 CSP에서 신규 고객에 대해 사용할 수 있게 되었습니다(아래 정의 참조).
Microsoft는 2020년 5월 4일에 신규 고객에 대해 Office 365 E1 및 Microsoft 365 Business Basic의 새로운 할인 프로모션을 도입합니다. 이 프로모션에서는 월 단위로 청구되는 연간 구독의 처음 6개월간 해당 제품을 무료로 제공합니다.

이 프로모션은 현재 Office 365 E1 평가판 제품을 대체하며, 다음과 같은 차이가 있습니다.

- 파트너는 고유한 6개월 평가판 SKU 대신 처음 6개월 동안 100% 할인을 적용하는 CSP 프로모션을 사용하여 Office 365 E1 또는 Microsoft 365 Business Basic SKU를 거래합니다. [CSP 프로모션에 대해 자세히 알아보세요](../promotions.md).
- 주문은 엄밀히 말하면 서면 정책을 통해 신규 고객으로만 제한되었던 Office 365 E1 평가판이 아닌 "신규 고객만"으로 제한됩니다. 
- SMB 고객의 경우 이 프로모션에 Microsoft 365 Business Basic이 포함됩니다. 따라서 파트너는 비즈니스용 Microsoft 365가 포함된 올바른 제품군에서 실제 사용자 수가 300명 미만인 고객을 시작할 수 있습니다.
현재 Office 365 E1 평가판(2020년 5월 4일까지 사용 가능)으로 라이선스가 부여된 고객은 현재 플랜을 그대로 유지합니다. 이 변경 사항은 순 신규 고객에게만 영향을 줍니다.

신규 고객의 정의

이 프로모션에 해당하는 신규 고객은 다음과 같습니다.

1. 순 신규 고객 테넌트
2. Microsoft 파트너 FAQ(자주 묻는 질문) 가이드에 정의된 제품에 대한 테넌트에 현재 결제된 구독이나 최근에 취소된(지난 30일 이내) 구독이 없는 기존 고객

 
### <a name="additional-resources"></a>추가 리소스

- 보안 원격 작업에 대해 자세히 알아보려면 [파트너 원격 작업 리소스 센터](https://www.microsoft.com/microsoft-365/partners/remotework)를 방문하세요.


### <a name="next-steps"></a>다음 단계

위의 변경 사항과 2020년 5월 가격표를 숙지하고 이 정보를 조직 내의 모든 해당 담당자와 공유하세요.

## <a name="azure-reservations-new-customer-self-serve-capability-with-partner-permission"></a><a id="13"/></a>Azure Reservations 파트너 권한이 포함된 새로운 고객 셀프 서비스 기능

### <a name="categories"></a>범주

- 날짜: 2020-04-29
- 다음을 입력합니다. 기능
- 상태: 지금 이용 가능
- [자세한 정보](https://partner.microsoft.com/resources/collection/customer-permission-to-purchase#/)

### <a name="impacted-audience"></a>영향을 받는 대상

클라우드 솔루션 공급자 프로그램의 파트너

### <a name="summary"></a>요약

Microsoft는 고객이 Microsoft와 파트너의 제품 및 서비스를 보다 쉽게 구입하고 사용할 수 있도록 유연성과 옵션을 제공하기 위해 노력하고 있습니다.

지속적인 노력의 일환으로 Microsoft는 고객의 구매 환경을 간소화하고 CSP의 파트너를 통한 Azure 예약을 더욱 향상시키는 Azure 예약에 대한 새로운 고객 셀프 서비스 기능을 도입합니다.

Azure 예약은 할인된 요금으로 Azure 서비스를 구입할 수 있는 좋은 방법입니다. 이 새로운 기능을 사용하여 파트너는 고객이 이미 구매한 기존 Azure 구독 내에서 유연하게 소유하고 있는 리소스를 구입할 수 있는 권한을 고객에게 제공할 수 있습니다.

기능의 작동 방식에 대한 자세한 내용은 다음과 같습니다.

- 파트너는 개별 고객 계정 수준에서 기능을 사용하도록 설정하고 고객에게 구매 관리 권한을 제공할 수 있습니다.
- 기본적으로 이 기능은 "off"로 설정되며 파트너가 이 기능을 사용하도록 설정하고 고객에게 해당 권한을 제공합니다.
- 파트너는 파트너 센터에서 고객의 구매를 확인하고 관리할 수 있습니다.
- 프로세스를 자동화하도록 API를 사용하여 규모에 맞게 고객 권한을 허용할 수 있습니다.

### <a name="next-steps"></a>다음 단계

1. 새 기능을 사용하여 고객이 소유하고 있는 Azure 예약을 구입할 수 있도록 지원합니다.
2. [관련 컬렉션에서 사용할 수 있는 리소스를 검토](https://partner.microsoft.com/resources/collection/customer-permission-to-purchase#/)하여 자세한 정보를 확인합니다. 여기에는 [기술 문서](../give-customers-permission.md), 파트너 환경 연습 및 고객 환경 연습이 포함되어 있습니다.

## <a name="microsoft-reversed-the-csp-eur-price-for-microsoft-365-e3"></a><a id="12"/></a>Microsoft는 Microsoft 365 E3의 CSP EUR 가격을 되돌림

### <a name="categories"></a>범주

- 날짜: 2020-04-24
- 다음을 입력합니다. 제품/시장
- 상태: 지금 이용 가능
- [자세한 정보](https://flow.microsoft.com/ui-flows/)

### <a name="summary"></a>요약

Microsoft는 2020년 4월 1일에 M365(Microsoft 365) E3의 EUR 가격 책정에서 오류를 수정했습니다. 그러나 2020년 4월 22일에 Microsoft는 관련 특별 상황에서 파트너를 지원하고자 가격을 일시적으로 되돌렸습니다.

### <a name="impacted-audience"></a>영향을 받는 대상

클라우드 솔루션 공급자 프로그램의 EUR 가격표를 통해 M365 E3을 거래하는 모든 파트너

### <a name="details"></a>세부 정보

2020년 초 CSP에서 Microsoft 365 E3의 EUR 가격 책정에서 정가가 실제 가격보다 낮은 오류를 발견했습니다. 2020년 4월 1일에 제품의 가격대를 수정하는 시스템 업데이트가 예정되었습니다.

가격 책정에 오류가 있더라도 특별한 상황을 고려하여 볼 때 이 시점에 오류를 수정하고 가격을 높이는 것은 적절하지 않습니다. 따라서 2020년 4월 22일부터 이전의 낮은 순 가격으로 일시적으로 돌아갑니다. 모든 송장에는 이전의 낮은 가격이 즉시 반영되지만, 가격표에는 향후 2020년 5월 가격표의 낮은 가격이 표시됩니다.

파트너는 2020년 4월 가격표에 나열된 M365 E3 EUR 가격 책정을 사용하면 안 됩니다. 대신 2020년 3월 가격표에 나열된 가격 책정을 참조하세요.

이 전례 없는 시간 동안 파트너 커뮤니티에서 보여주는 노고에 감사드립니다.

### <a name="next-steps"></a>다음 단계

이 정보를 조직 내의 모든 해당 담당자와 공유하세요.

### <a name="questions"></a>질문이 있으세요?

Microsoft와의 작업과 관련하여 질문, 쿼리 또는 자세한 내용을 보려면 지원에 문의하세요.

## <a name="reminder-activation-of-security-safeguards-for-partner-center-transactions-in-csp"></a><a id="11"/></a>미리 알림: CSP의 파트너 센터 트랜잭션에 대한 보안 세이프가드 활성화

### <a name="categories"></a>범주

- 날짜: 2020-04-23
- 다음을 입력합니다. 기능
- 상태: 출시 예정
- [자세한 정보](https://partner.microsoft.com/resources/collection/partner-security-requirements#/)

### <a name="target-partner-audiences"></a>대상 파트너 그룹

- CSP(클라우드 솔루션 공급자) 프로그램에 참여하는 모든 파트너
- 모든 CPV(제어판 공급업체)
- 모든 Advisor 파트너

### <a name="summary"></a>요약

더 강력하고 지속적인 보안 및 개인 정보 보호가 최우선 순위이며, 파트너가 고객과 테넌트를 보호할 수 있도록 지속적으로 지원하고 있습니다. Microsoft는 2019년에 [필수 보안 요구 사항](https://blogs.partner.microsoft.com/mpn/safeguard-business-security-best-practices/)을 도입했습니다. 클라우드 솔루션 공급자 프로그램에 참여하는 모든 파트너, 제어판 공급업체 및 Advisor는 규정 준수 상태를 유지하기 위한 요구 사항을 구현해야 합니다.

Microsoft는 최근 모든 파트너 테넌트에 대해 AOBO(관리자 위임) 기능의 보안 세이프가드 활성화를 성공적으로 완료하여 ID 도난 관련 인시던트로부터 파트너의 비즈니스 및 고객을 보호합니다.

파트너와 고객에 대한 방어를 한층 더 강화하기 위해 **2020년 5월 1일부터 CSP의 파트너 센터 트랜잭션에 대해 세이프가드 활성화가 시작됩니다**. 추가 보안 세이프가드를 활성화하면 파트너 센터 사용자 환경 또는 API를 사용한 CSP의 모든 트랜잭션에 대해 MFA(다단계 인증)를 사용하도록 파트너에게 요청하여 무단 액세스를 방지하고 ID 도난 관련 보안 인시던트로부터 고객과 테넌트를 보호할 수 있게 됩니다. 자세한 내용은 [여기](../partner-security-requirements-mandating-mfa.md)를 참조하세요.

### <a name="next-steps"></a>다음 단계

- **테넌트 및 사용자를 보호하기 위해 권장되는 보안 요구 사항을 활성화하지 않은 경우** 즉시 활성화하여 규정 준수 상태를 유지하고 CSP에서 고객을 관리하거나 거래할 때 비즈니스 중단이 발생하지 않도록 해야 합니다. 보안 요구 사항을 구현하지 않는 파트너가 CSP 프로그램에 참여하면 위험에 노출될 수 있습니다.
- 간접 공급자는 관련 이메일 템플릿을 사용하여 각자의 비즈니스에 대해 이 보안 요구 사항을 구현하고 재판매인과 협력하여 재판매인도 이 요구 사항을 구현하도록 해야 합니다. 또한 이메일 통신을 통해 재판매인에게 직접 연락합니다.
- [파트너 센터 보안 요구 사항 상태 페이지](https://partner.microsoft.com/commerce/security/compliance)를 방문하여 진행 상황과 수행할 주요 작업을 확인하세요. 전역 관리자와 협력하여 각 테넌트의 보안 요구 사항 진행 상황을 정기적으로 확인하세요.
- CPV(제어판 공급업체) 솔루션을 사용 중인 경우 CPV에게 문의하세요.

### <a name="additional-resources"></a>추가 리소스

보안 요구 사항을 구현하는 데 도움이 되도록 다음 리소스를 검토하세요.

- 파트너 보안 요구 사항을 구현하려면 [단계별 가이드](../partner-security-requirements.md)를 확인합니다.
- 파트너 보안 요구 사항에 관해 [자주 묻는 질문](../partner-security-requirements-faq.yml)과 대답을 참조합니다.
- [보안 요구 사항 상태 확인 가이드](../partner-security-compliance.md)를 살펴봅니다.
- [파트너 보안 요구 사항 구현](https://partner.microsoft.com/resources/collection/partner-security-requirements#/) 컬렉션의 추가 리소스에 액세스합니다.
- [파트너 센터 보안 지침 커뮤니티 그룹](https://aka.ms/MPCSecurityGuidance)에 책갈피를 지정합니다.

기술 문제에 대한 추가 지원이 필요한 경우 [지원 티켓을 제출](https://partner.microsoft.com/dashboard/support/csp/servicerequests/create?topicid=32674991)하세요. 이 보안 요구 사항을 구현하기 위한 조치를 취해야 할 조직의 다른 개인에게 이 이메일을 전달하세요.

파트너십과 헌신을 통해 Microsoft 에코시스템이 신뢰를 바탕으로 운영되는 데 도움을 주셔서 감사합니다.

>[!NOTE] 
>기준 정책을 사용 중이며 보안 기본값 정책이나 [다른 MFA 구현 옵션](..\partner-security-requirements.md#implementing-multi-factor-authentication)으로 전환하지 않은 경우 [여기에 있는 주요 리소스](https://partner.microsoft.com/resources/collection/security-defaults-enable-mfa#/)를 참조하여 최대한 빨리 전환하세요.

최소 50개의 라이선스를 보유한 Microsoft Defender ATP 고객은 월별 서버 운영 체제 환경당 $5의 가격으로 서버 엔드포인트 보호에 새로운 Microsoft Defender ATP for Servers SKU를 사용할 수 있습니다. 서버의 향상된 클라우드 워크로드 보호를 위해 Microsoft Defender ATP for Servers SKU 고객은 매월 서버 OSE당 추가 $10로 Azure Security Center Standard SKU로 업그레이드하고 적응형 애플리케이션 제어, Just-In-Time 가상 머신 액세스, 파일 무결성 모니터링, Azure 리소스를 보호하기 위한 통합 환경 등의 기능과 향후 Azure Arc와의 통합 같은 추가 가치에서 이점을 얻을 수 있습니다.

현재 Azure Security Standard에 대한 변경 사항은 없습니다. Azure Security Center Standard는 서버 가상 머신 엔드포인트뿐만 아니라 IoT, Azure Kubernetes Services(미리 보기), Azure Container Registry(미리 보기), Key Vault(미리 보기), App Services, SQL, Storage 등의 다양한 Azure 리소스를 보호하기 위한 통합 사용자 환경을 계속 제공합니다.

ASC Standard 기능에서 이점을 얻으려면 MDATP for Servers SKU 소유자는 $10 버전 업그레이드 SKU를 통해 ASC(Azure Security Center) Standard로 업그레이드해야 합니다.​

## <a name="now-available---operations-readiness-resources-in-one-place"></a><a id="3"/></a>지금 이용 가능 - 한곳에서 제공되는 Operations Readiness 리소스

이 새로운 환경과 앞으로 몇 주에 걸쳐 계속 개선될 일부 영역에 대한 자세한 내용은 [업데이트된 FAQ](https://assetsprod.microsoft.com/mpn/readiness-in-one-place-partner-login-faq.pdf)(여기)를 검토하세요.

**여러분의 피드백은 매우 중요합니다!** 새 Operations Readiness 리소스 위치를 사용하면서 계속해서 환경을 개선할 수 있도록 피드백을 남겨 주세요. 각 페이지 아래쪽에서 사용할 수 있는 **이 페이지가 도움이 되었나요? 예 | 아니요** 피드백 도구를 사용하여 피드백을 남길 수 있습니다.

Microsoft는 모든 Microsoft 프로그램과 관련하여 커뮤니케이션하고 준비 자료를 제공하는 방식을 간소화합니다. [CSP 프로그램 업데이트 리소스 갤러리](https://partner.microsoft.com/resources#/)와 [Microsoft Readiness 사이트](https://www.microsoftreadiness.com/)의 자산이 결합된 단일 Operations Readiness 리소스 위치가 Microsoft 파트너 웹 사이트([https://partner.microsoft.com](https://partner.microsoft.com))에 도입됩니다.

작업 준비 리소스가 한곳으로 통합되면 파트너가 최신 업데이트와 중요 공지 사항, 모든 Microsoft 채널 및 프로그램의 주요 작업 준비 리소스를 그 어느 때보다 쉽게 찾을 수 있게 됩니다. Single Sign-On 기능과 향상된 검색, 필터, 정렬 기능으로 각종 자료에 손쉽게 액세스하고 이용하며 Microsoft와 편리한 방식으로 비즈니스 거래를 수행할 수 있습니다.

### <a name="expanding-on-the-success-of-the-csp-program-updates-resource-gallery"></a>CSP 프로그램 업데이트 리소스 갤러리와 유사한 구조:

partner.microsoft.com의 작업 준비 리소스 사이트는 기존의 CSP 프로그램 업데이트 리소스 갤러리와 비슷하게 기능하며, 다음과 같은 개선 사항 및 수정 사항이 반영됩니다.

- **프로그램 페이지:** 각 페이지는 해당 프로그램의 콘텐츠만 표시하도록 미리 필터가 적용되기 때문에 해당 프로그램의 최근 문서 업데이트를 쉽게 찾을 수 있게 됩니다.
- **강력한 갤러리 검색:** 모든 프로그램과 콘텐츠에서 빠르고 간편하게 콘텐츠를 찾을 수 있도록 도와주는 콘텐츠 필터가 제공됩니다(예: CSP).
- **알림:** 각 페이지는 해당 프로그램의 콘텐츠만 표시하도록 미리 필터가 적용되기 때문에 해당 프로그램의 최신 공지 사항과 최근 문서 업데이트를 쉽게 찾을 수 있게 됩니다. CSP를 위한 프로그램 페이지도 제공되지만, CSP 공지는 계속해서 파트너 센터에 게시됩니다.
- **새 위치:** partner.microsoft.com의 새로운 작업 리소스 위치가 정식으로 공개되면 Microsoft Readiness 사이트와 CSP 프로그램 업데이트 리소스 갤러리에는 더 이상 액세스할 수 없으며 새로운 작업 리소스 사이트로 리디렉션됩니다. 파트너는 콘텐츠 프로그램 필터를 사용하여 CSP 또는 다른 프로그램 리소스로 필터링할 수 있습니다.
- **확장된 콘텐츠:** Microsoft Readiness 사이트와 CSP 프로그램 업데이트 리소스 갤러리의 모든 최근 및 관련 콘텐츠가 새로운 작업 준비 사이트로 이전됩니다. 즉, 이제 더 많은 프로그램의 콘텐츠가 한곳에 정리됩니다.

## <a name="microsoft-365-business-voice-available-to-csp-partners-from-april-1-2020"></a><a id="2"/></a>2020년 4월 1일부터 CSP 파트너에게 Microsoft 365 Business Voice가 제공됨

### <a name="categories"></a>범주

- 날짜: 2020-04-01
- 다음을 입력합니다. 제품/시장
- 상태: 지금 이용 가능
- [자세한 정보](https://partner.microsoft.com/resources/collection/microsoft_365_business_voice_in_csp#/)

### <a name="summary"></a>요약

중소기업용으로 빌드된 클라우드 기반 전화 시스템인 Microsoft 365 Business Voice는 2020년 4월 1일부터 여기에 나열된 국가에서 사용할 수 있습니다.

### <a name="impacted-audience"></a>영향을 받는 대상

- 여기에 나열된 [국가](https://partner.microsoft.com/resources/collection/microsoft_365_business_voice_in_csp#/)에서 운영하는 모든 CSP 파트너.
- 오스트레일리아의 파트너는 아래 지침을 확인해야 합니다.

### <a name="details"></a>세부 정보

Microsoft는 2020년 4월 1일에 추가 지역에 Microsoft 365 Business Voice를 도입합니다. Microsoft 365 Business Voice와 통화 플랜 불포함 Microsoft 365 Business Voice(직접 라우팅 사용)의 두 가지 제품을 사용할 수 있습니다. 두 제품 모두 미국에서 제공될 예정입니다.

### <a name="customer-value-proposition"></a>고객 가치 제안

Microsoft 365 Business Voice는 최대 300명의 사용자를 보유한 기업을 위한 클라우드 기반 전화 시스템입니다. Microsoft Teams의 전화 번호로 전화를 걸고 받을 수 있는 기능을 제공합니다. Teams와 함께 통화, 채팅 및 모임을 결합하는 올인원 통신 솔루션을 제공합니다.

Microsoft 365 Business Voice는 컴퓨터, 스마트폰, 일반 전화기 등 업무가 수행되는 모든 곳에서 작동하는 단일 애플리케이션으로 생산성과 통신을 단순화 및 간소화합니다. Microsoft 365 Business Voice의 이점은 다음과 같습니다.

- 올인원 통신. 통화, 채팅, 모임, 일정 및 이메일을 통합하여 업무 수행 방식을 혁신합니다.
- 손쉬운 인텔리전트 통화. 인텔리전스가 적용되어 더 유연하고 손쉽게 통신하므로 연결된 상태를 유지할 수 있습니다.
- Microsoft 클라우드에서 제공됨. 안정적이고 안전한 클라우드 기반 솔루션을 통해 자신감을 가지고 작업할 수 있습니다.

Microsoft 365 Business Voice는 다음 플랜에 대한 추가 기능으로 사용할 수 있습니다.

- Office 365 Business Essentials, Business Premium, F1, A1, E1, A3, E3
- Microsoft 365 Business, F1, A3, E3

### <a name="partner-value-proposition"></a>파트너 가치 제안

파트너는 다음과 같은 방법으로 이 기회에서 수익을 창출할 수 있습니다.

- 통화 큐/자동 전화 교환이 포함된 전화 시스템을 사용하고, 네트워크 품질을 평가하거나, 직접 라우팅 및 오디오 회의를 구성합니다.
- 사용자 채택/교육 서비스를 제공하여 조직에서 확실히 준비하도록 하고 최신 통신 도구인 Microsoft Teams 사용을 시작합니다.
- 품질 보고, 디바이스 관리, 사용자 지원 등의 관리형 서비스를 통해 고객 환경을 지원하고 모니터링합니다.

**Microsoft 365 Business Voice(미국, 캐나다, 영국)**

2019년 11월 영국 및 캐나다에서의 출시에 이어, 2020년 4월 1일에 통화 플랜 포함 Business Voice를 미국에서 구매할 수 있습니다.

국제 통화 기능을 원하는 고객을 위해 추가 기능 제품도 제공됩니다.

- Microsoft 365 Business Voice: 9f9f2c7b-c961-402b-9421-8e3c9207eeb3
- SMB용 Microsoft 365 국제 통화 플랜: c01a89e5-d246-42d6-b21a-f4a1e9b04f56

**통화 플랜 불포함 Microsoft 365 Business Voice(70개 이상 국가)**

또는 2020년 4월 1일부터 전화 시스템 및 오디오 회의가 포함된 통화 플랜 불포함 Business Voice 제품을 구매할 수 있습니다. 이 제품은 여기에 나열된 모든 국가에서 사용할 수 있습니다.

- **미국만 해당:** 통화 플랜 불포함 Microsoft 365 Business Voice: 39d77d0f-eb8f-4ebc-b618-692e1cc59c8f
- **기타 국가:** 통화 플랜 불포함 Microsoft 365 Business Voice: cc69a07c-8c51-457f-bb2a-f21a62d6bede

이 제품에서 혜택을 얻는 방법을 이해하려면 현지 자회사와 소통하세요.

### <a name="additional-guidance-for-partners-in-the-united-states-and-puerto-rico"></a>미국 및 푸에르토리코의 파트너에 대한 추가 지침

미국 및 푸에르토리코에서 Business Voice는 세금이 포함되어 있습니다. 파트너에게 청구되고 파트너가 Microsoft에 지불하는 가격에는 관련 미국 통신세 및 요금이 포함됩니다. 다른 시장에서는 일반적인 청구 프로세스에 따라 세금이 적용됩니다. Business Voice 가격 인상을 결정하기 전에 세금 송금에 관해 변호사에게 문의하세요. Business Voice 가격이 인상되면 파트너는 통신세를 송금하는 방법이 필요합니다. 통신세를 송금하는 방법이 마련되지 않으면 파트너는 가격 인상에 대한 인상된 세금을 지불하지 않으므로 세금 위반이 발생합니다. Microsoft는 미국에서 Business Voice에 대해 파트너에게 추가 CSP 인센티브를 제공하여 잠재적 이익 손실을 보상합니다.

### <a name="guidance-for-partners-in-australia"></a>오스트레일리아의 파트너를 위한 지침

오스트레일리아의 파트너는 2020년 3월의 미리 보기 가격표에 통화 플랜 불포함 Microsoft 365 Business Voice가 표시되어 있긴 하지만, 2020년 4월 1일의 프로덕션 가격표에서 통화 플랜 불포함 Microsoft 365 Business Voice 제품을 사용할 수 없다는 점에 유의해야 합니다. 이점 양해해 주시기 바랍니다.

### <a name="further-countries"></a>추가 국가

앞으로 점차 Office 365 및 Microsoft 365를 사용할 수 있는 모든 국가에 Microsoft 365 Business Voice가 제공되기를 기대합니다.

## <a name="act-now-partners-are-now-required-to-have-accepted-the-microsoft-partner-agreement"></a><a id="1"/></a>지금 필요한 작업: 이제 파트너는 Microsoft 파트너 계약에 동의해야 합니다.

### <a name="categories"></a>범주

- 날짜: 2020-04-01
- 다음을 입력합니다. 기능
- 상태: 지금 이용 가능
- [자세한 정보](https://partner.microsoft.com/resources/collection/new-microsoft-cloud-agreement-csp)

### <a name="impacted-audience"></a>영향을 받는 대상
모든 CSP 파트너.

**2020년 4월 1일 업데이트** 2020년 1월 31일까지 CSP(클라우드 솔루션 공급자) 프로그램의 모든 파트너는 거래하는 모든 CSP 테넌트에 대한 계약에 동의해야 합니다. 이전에 계약의 다음 구현 단계가 2020년 4월에 시작된다고 알려드렸습니다.  비즈니스 중단을 방지하기 위해 다음 구현 단계를 2020년 4월 이후로 연장하기로 결정했습니다. 클라우드 솔루션 공급자 파트너 중 일부에게 MPA에 동의하는 데 문제가 있다는 피드백을 받았습니다. 이러한 상황에 따라 계약에 동의하는 데 문제가 있는 파트너를 지원하기 위한 Microsoft 리소스를 마련하기 위해 노력하고 있습니다.

### <a name="summary"></a>요약
2020년 1월 31일부터 클라우드 솔루션 공급자 프로그램의 모든 파트너(간접 재판매인 포함)는 Microsoft 파트너 계약에 동의해야 중단 없이 신규 고객을 창출하거나 CSP 프로그램의 기존 고객을 위해 새로 주문할 수 있습니다. 이제 CSP 프로그램의 모든 파트너에게 Microsoft 파트너 계약 동의는 필수입니다.

2020년 4월부터:
- Microsoft 파트너 계약에 동의하지 않은 직접 청구 파트너 및 간접 공급자는 새로 주문할 수 없으며 기존 구독 관리만 가능합니다.
- Microsoft 파트너 계약에 동의하지 않은 간접 재판매인은 더 이상 새 주문에 대해 공식 파트너 역할을 할 수 없습니다.

**다음 작업을 아직 수행하지 않은 경우 즉시 완료하세요.**

### <a name="actions-for-direct-bill-partners"></a>직접 청구 파트너가 수행할 작업:
1. 조직의 주요 관련자에게 Microsoft 파트너 계약을 검토하도록 알립니다.
2. 전역 관리자 자격 증명을 사용하여 [파트너 센터의 CSP 개요 페이지](https://partner.microsoft.com/pcv/dashboard/overview)에서 Microsoft 파트너 계약에 동의합니다.

추가 정보 및 리소스는 CSP 갤러리 페이지([여기](https://partner.microsoft.com/resources/collection/new-microsoft-cloud-agreement-csp#/))를 통해 사용할 수 있습니다.

### <a name="actions-for-indirect-providers"></a>간접 공급자가 수행할 작업
1. 조직의 주요 관련자에게 Microsoft 파트너 계약을 검토하도록 알립니다.
2. 전역 관리자 자격 증명을 사용하여 [파트너 센터의 CSP 개요 페이지](https://partner.microsoft.com/pcv/dashboard/overview)에서 Microsoft 파트너 계약에 동의합니다.
3. [API를 사용하여 간접 재판매인의 서명 상태를 확인합니다](/partner-center/develop/verify-indirect-reseller-mpa-status).
4. 아래 작업을 완료하도록 간접 재판매인을 지원합니다.

추가 정보 및 리소스는 CSP 갤러리 페이지([여기](https://partner.microsoft.com/resources/collection/new-microsoft-cloud-agreement-csp#/))를 통해 사용할 수 있습니다.

Microsoft는 간접 공급자가 거래 적용 기능을 테스트하는 중에 발생하는 기술적인 문제에 대해 알고 있습니다. 이 문제는 2020년 4월까지 해결될 예정입니다.

### <a name="actions-for-indirect-resellers"></a>간접 재판매인이 수행할 작업
1. 간접 재판매인으로 [파트너 센터에 온보딩](https://partner.microsoft.com/resources/collection/indirect-reseller-onboarding-for-mpa-in-csp#/)하고 온보딩 프로세스의 일부로 Microsoft 파트너 계약에 동의합니다.
2. 이전에 간접 재판매인으로 파트너 센터에 온보딩한 경우 전역 관리자 자격 증명을 사용하여 [파트너 센터 대시보드 페이지](https://partner.microsoft.com/pcv/dashboard/overview)에서 Microsoft 파트너 계약에 동의합니다.
간접 재판매인 관련 정보 및 리소스는 재판매인 CSP 갤러리 페이지([여기](https://partner.microsoft.com/resources/collection/indirect-reseller-onboarding-for-mpa-in-csp#/))를 통해 사용할 수 있습니다.
