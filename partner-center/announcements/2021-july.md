# <a name="july-2021-announcements"></a>2021년 7월 공지

이 페이지에서는 2021년 7월 Microsoft 파트너 센터에 대한 공지를 제공합니다.

## <a name="software-in-microsoft-china-corporation-limited-mccl-cloud-solution-provider-csp-and-open-license-change"></a><a name="9"></a>MCCL(Microsoft China Corporation Limited) CSP(클라우드 솔루션 공급자)의 소프트웨어 및 오픈 라이선스 변경

### <a name="categories"></a>범주

- 날짜: 2021-07-13
- 기능
 
### <a name="summary"></a>요약

[2021년 5월 6일](https://partner.microsoft.com/resources/detail/csp-perpetual-software-coming-to-china-pdf) 업데이트 이후 MCCL(Microsoft China Corporation Limited) CSP(클라우드 솔루션 공급자)의 소프트웨어 랜딩 및 공개 프로그램 변경 사항에 대한 주요 공지의 진행 상황을 알려드립니다.

### <a name="impacted-audience"></a>영향을 받는 대상

MCCL CSP 프로그램의 파트너

### <a name="details"></a>세부 정보

#### <a name="software-in-mccl-csp-general-availability-date"></a>MCCL CSP의 소프트웨어 일반 공급 날짜 

2021년 8월 첫째 주를 목표로 MCCL CSP 프로그램에 등록한 파트너는 MCCL CSP를 통해 온-프레미스 배포용 영구 소프트웨어 라이선스를 판매할 수 있습니다. 이를 통해 디지털 혁신의 일환으로 필요한 라이선스 유연성이 고객에게 제공됩니다.

#### <a name="open-license-program-retirement"></a>오픈 라이선스 프로그램 사용 중지

2022년 1월 1일부터 고객은 더 이상 Microsoft 오픈 라이선스 프로그램을 통해 신규 소프트웨어 라이선스를 구매하거나 갱신할 수 없습니다. 그때까지 모든 고객은 오픈 라이선스 프로그램을 통해 계속 구매할 수 있습니다.

#### <a name="open-license-program-price-change-for-government"></a>정부용 오픈 라이선스 프로그램 가격 변경

2021년 7월부터 정부 고객용 L(License), SA(Software Assurance), L&SA(Microsoft License and Software Assurance)와 같은 오픈 라이선스 프로그램 정부 온-프레미스 SKU의 가격이 다양하게 인상됩니다. 이는 정부 기관에 대한 가격 설정을 새로운 상거래 환경에 맞추기 위해 2018년부터 진행되어 온 작업입니다. 그러면 오픈 라이선스 정부 가격은 최저 상업 가격과 동일해져서 CSP 프로그램의 새로운 상거래 환경 가격과 동등해지고 정부 고객이 솔루션을 더 쉽게 비교할 수 있습니다. 오픈 라이선스 정부 가격 책정 [FAQ](https://aka.ms/openlicensegovpricefaq) 및 [L100 데크](https://aka.ms/openlicensegovpriceL100)에서 자세히 알아보세요.

#### <a name="sa-purchases"></a>SA 구매

SA 혜택은 만료일이 2022년 1월 1일 이후라도 SA 기간이 만료될 때까지 계속됩니다. Microsoft OV(Open Value) 프로그램은 SA를 포함한 향후 구매에 권장됩니다.

>[!NOTE]
>OV 또는 Open Value 구독 프로그램을 변경할 계획은 없습니다.

## <a name="readiness-changes-to-the-cloud-solution-provider-csp-customer-address-validation-api-to-go-live-on-july-13-2021"></a><a name="8"></a>준비: 2021년 7월 13일에 적용되는 CSP(클라우드 솔루션 공급자) 고객 주소 유효성 검사 API에 대한 변경 사항

### <a name="summary"></a>요약

주소 유효성 검사 API에 대한 변경 사항은 2021년 7월 13일부터 적용됩니다.

### <a name="categories"></a>범주

- 날짜: 2021-07-09
- 준비

### <a name="impacted-audience"></a>영향을 받는 대상

새 고객 주소 정보를 만들거나 기존 고객 주소 정보를 업데이트하는 CSP(클라우드 솔루션 공급자) 직접 청구 파트너 및 간접 공급자

### <a name="details"></a>세부 정보

Microsoft는 신뢰할 수 있는 방식으로 운영되며, 규정을 준수하고 안전하며 보안이 유지되는 고객 주소 유효성 검사 방법을 제공하여 CSP 프로그램에서 고객 구독을 거래할 수 있도록 최선을 다하고 있습니다. 2021년 7월 13일부터 주소 유효성 검사 API에 대한 변경 사항이 적용됩니다.
 
이번 변경 사항은 주소 유효성 검사 API에만 적용됩니다. 고객 만들기 및 청구 프로필 업데이트 API는 영향을 받지 않습니다. 현재 제안된 주소를 고객 만들기 API와 함께 사용할 필요는 없지만 매우 권장되는 주소입니다.

응답에서 다음 상태 메시지 중 하나를 반환합니다.

| 상태 | Description | 반환되는 추천 주소 수 |
|------------|---------------|------------------|
| 배송 가능 확인됨  | 주소가 확인되어 제품을 배송할 수 있습니다.  | 한 개  |
| 확인됨  | 주소가 확인되었습니다.  | 한 개  |
| 상호 작용 필요  | 추천 주소가 많이 변경되어 사용자 확인이 필요합니다.  | 한 개  |
| 거리명 부분  | 주소에 입력된 거리명이 완전하지 않아 추가 정보가 필요합니다.  | 여러 개—최대 3개  |
| 부지 부분  | 입력된 부지(건물 번호, 다세대 동호수 등)가 완전하지 않아 추가 정보가 필요합니다.  | 여러 개—최대 3개  |
| 여러 개  | 주소에서 완전하지 않은 필드가 여러 개 있습니다(거리명 부분과 부지 부분 포함).  | 여러 개—최대 3개  |
| 없음  | 주소가 올바르지 않습니다.  | 없음  |
| 확인 안 함  | 유효성 검사 프로세스를 통해 주소를 보낼 수 없습니다.   | 없음  |

주소를 확인하기 위해 주소 유효성 검사 API를 통해 주소를 제출하면 다음과 같은 응답 스키마가 반환됩니다.

```csharp

// <summary>
/// Object represents the address validation response.
/// </summary>

public class AddressValidationResponse
{
   /// <summary>
   /// Gets or sets the original address
   /// </summary>
   /// <value>
   /// Original Address
   /// </value>
   public Address OriginalAddress { get; set; }

   /// <summary>
   /// Gets or sets the suggested addresses
   /// </summary>
   /// <value>
   /// Suggested Addresses
   /// </value>
   public List<Address> SuggestedAddresses { get; set; }

   /// <summary>
   /// Gets or sets the validation status
   /// </summary>
   /// <value>
   /// Status
   /// </value>
   public string Status { get; set; }

   /// <summary>
   /// Gets or sets the validation message
   /// </summary>
   /// <value>
   /// Validation Message
   /// </value>
   public string ValidationMessage { get; set; }

```

미국의 경우 우편 번호에 5자리 숫자만 입력하면 우편 번호 줄에 4자리 숫자 접미사가 추가된 응답이 반환됩니다.
다음은 샘플 응답입니다.

```C#

"suggestedAddress": [
{ 
"country": "US", 
"state": "WA", 
"city": "Redmond", 
"addressLine1": "1 Microsoft Way", 
"postalCode": "98052-8300" 
}
], 
"originalAddress": { 
"country": "US", 
"city": "Redmond", 
"addressLine1": "1 Micro Way", 
"postalCode": "98052" 
}, 
"status": "InteractionRequired", 
"validationMessage": "Address field invalid for property: ‘Street’" 
}
```

## <a name="keep-your-microsoft-certifications-active-with-help-from-partner-center"></a><a name="6"></a>파트너 센터의 도움을 받아 Microsoft 인증을 활성 상태로 유지

### <a name="categories"></a>범주

- 날짜: 2021-07-09
- 비즈니스 확장

### <a name="impacted-audience"></a>영향을 받는 대상

Microsoft 인증 상태 및 갱신을 관리하는 파트너 관리자 및 전역 관리자

### <a name="details"></a>세부 정보

파트너 센터의 도움을 받아 Microsoft 인증을 활성 상태로 쉽게 유지하세요.

파트너 센터 대시보드에서 항상 조직의 인증을 볼 수 있었지만 이제는 사용자 기술 보고서와 파트너 센터 인사이트 교육 보고서 모두에서 만료 날짜도 볼 수 있습니다. 이러한 업데이트를 통해 조직의 인증 갱신 프로세스를 제어할 수 있습니다. 보고 업데이트를 찾는 방법과 이러한 업데이트가 어떤 혜택을 제공하는지 알아보려면 읽어보세요.

파트너 센터 대시보드의 역량 섹션에서 역량 보고서를 찾을 수 있습니다. 데이터를 Excel로 내보내면 사용자 기술 보고서가 나타나며, 여기에는 역량을 갖추기 위해 획득한 업적(예: 인증 및 시험) 목록이 표시됩니다. 보고서 끝에 "기술 만료 날짜"라는 새로운 열이 있는데, 이는 직원의 역할 기반 및 전문성 인증 만료 날짜를 나타냅니다.  

파트너 센터 대시보드의 조직 기술 섹션에서 파트너 센터 인사이트 교육 보고서를 찾을 수 있습니다. 이 보고서의 데이터를 Excel로 내보내면 팀 인증의 만료 날짜가 포함된 새로운 만료 날짜 열을 찾을 수 있습니다.

이러한 업데이트를 통해 신속하게 조치를 취하고 직원에게 인증을 갱신하도록 알려주어 그들의 자격 증명을 유지하고 MPN 프로그램 상태를 활성 상태로 유지할 수 있습니다.

새로운 인증 갱신 프로세스는 어떻게 작동하나요?  

- 테스트 센터에 가서 인증 시험을 재응시하기 위해 비용을 지불하는 대신 직원이 Microsoft Learn을 방문하여 최신 기술 업데이트에 대해 테스트하는 무료 인증 갱신 평가를 통과하게 됩니다.
- 인증 만료 날짜가 다가오면 Microsoft로부터 이메일 알림을 받아 Microsoft Learn의 인증 갱신 평가로 이동합니다.
- 온라인 학습 경로에는 평가를 준비하기 위한 학습 자료가 포함되어 있으며 필요한 만큼 여러 번 시도할 수 있습니다.
- 평가에 합격하면 인증 만료가 1년 연장됩니다.
- 이 프로세스는 매년 반복되어 기술이 최신 상태가 되도록 합니다.

이 새로운 연간 갱신 프로세스에 맞추기 위해 Microsoft는 역할 기반 및 전문 인증의 유효 기간도 2년에서 1년으로 변경합니다. 2021년 6월 30일부터 취득한 모든 역할 기반 및 전문 인증은 1년간 유효합니다.

Microsoft Fundamentals 및 이전 레거시 인증은 면제되며 새 프로세스의 영향을 받지 않습니다.

### <a name="next-steps"></a>다음 단계

1. 팀 인증 만료 날짜를 미리 확인하세요. 사용자 기술 보고서 또는 파트너 센터 인사이트 교육 보고서를 검토하여 곧 다가오는 인증 만료 날짜를 확인하세요.
2. 조치: 만료 날짜 전에 팀이 Microsoft 인증을 갱신하도록 권장합니다.

## <a name="dynamics-365-cloud-migration-promotion-and-retirement-of-discounted-offers"></a><a name="1"></a>Dynamics 365 클라우드 마이그레이션 프로모션 및 할인된 제품 폐기

### <a name="categories"></a>범주

- 날짜: 2021-07-01
- 기능

### <a name="summary"></a>요약

Microsoft는 2021년 8월 1일에 Dynamics 365 클라우드 마이그레이션 프로모션의 시작을 발표하게 되어 기쁘게 생각합니다.

### <a name="impacted-audience"></a>영향을 받는 대상

CSP(클라우드 솔루션 공급자) 파트너, Dynamics 온-프레미스 파트너 및 LSP

### <a name="details"></a>세부 정보

파트너는 CSP 프로그램을 통해 업계 최고의 클라우드 기반 솔루션을 사용하여 고객의 비즈니스를 디지털 방식으로 변환할 수 있습니다. 이러한 노력의 일환으로 2021년 7월 31일에 기존의 할인된 SKU(Dynamics 365 클라우드 마이그레이션 프로모션 및 [ 사용 중지된 SKU Excel 파일](https://partner.microsoft.com/resources/detail/dynamics-365-cloud-promotion-retired-skus-xls)의 사용 중지된 SKU 탭 참조)를 사용 중지할 예정이며 이를 새로운 Dynamics 365 클라우드 마이그레이션 프로모션 SKU로 대체하고 있습니다.

Microsoft는 2021년 8월 1일에 Dynamics 365 클라우드 마이그레이션 프로모션의 시작을 발표하게 되어 기쁘게 생각합니다. 이 새로운 프로모션은 고객이 클라우드로의 마이그레이션을 추진하는 데 할인된 Dynamics 365 온라인 서비스를 제공합니다. 또한 레거시 마이그레이션 제품에 따라 디지털 전환 과정을 시작하여 새로운 프로모션 가격 책정으로 전환하는 고객을 지원합니다.

#### <a name="dynamics-365-cloud-migration-promotion"></a>Dynamics 365 클라우드 마이그레이션 프로모션

**이점**

클라우드 마이그레이션 프로모션은 고객에게 다음과 같은 이점을 제공합니다.  

- 3년 구독의 Dynamics 365 온라인 서비스에 대한 40% 할인으로 클라우드로 마이그레이션하는 동안 가격 예측 가능성을 제공합니다.

- 클라우드 마이그레이션이 완료되면 온-프레미스 솔루션 지원 종료 전에 사용자 액세스를 지원하는 이중 액세스 권한(동시 온-프레미스 및 클라우드 사용)

- 단계별 클라우드 마이그레이션을 지원하여 조직에서 프로모션 기간 동안 새 사용자를 추가할 수 있습니다.

**용어**

라이선싱 프로그램: CSP

제품: Dynamics 365 클라우드 마이그레이션 프로모션

프로모션 할인: 40%

프로모션 기간: 2021년 8월 1일에서 2023년 6월 30일

구독 기간: 3년 구독, 프로모션 가격은 갱신되지 않습니다.

라이선싱 요구 사항:

- 활성 Dynamics 고객 서비스 및 지원 계획(예: 향상된 계획 또는 혜택 계획) 또는 SA(소프트웨어 보증) 및 Dynamics 365 온라인 서비스로 갱신하는 모든 Dynamics 온-프레미스 라이선스(Dynamics AX, CRM, 정부 파트너, 탐색, 구독 라이선스 및 기타)

- SA 또는 Dynamics 가격표의 Dynamics 365, Dynamics 365 클라우드 추가 기능, Dynamics 365 정규화된 제품 또는 Dynamics 365 클라우드 마이그레이션 제품에 대한 활성 구독이 있는 모든 Dynamics 365 온라인 서비스

- 이 프로모션은 순 신규 클라우드 고객 또는 비활성 Dynamics 온-프레미스 고객에게 적용되지 않습니다. ([온라인 서비스 정책](https://www.microsoft.com/licensing/terms/productoffering/MicrosoftDynamics365Services/EAEAS)을 참조하세요.)

**SKU**

SKU의 전체 목록은 Dynamics 365 클라우드 마이그레이션 프로모션 및 [사용 중지된 SKU Excel 파일](https://partner.microsoft.com/resources/detail/dynamics-365-cloud-promotion-retired-skus-xls))에서 프로모션 SKU 탭을 참조하세요.
