## <a name="now-available-windows-7-year-2-esu-for-cloud-solution-provider-csp"></a><a name="6"></a>이제 CSP(클라우드 솔루션 공급자)에 사용할 수 있는 Windows 7 Year 2 ESU

### <a name="categories"></a>범주

- 날짜: 2020-06-29
- 다음을 입력합니다. 제품/시장
- 상태: 지금 이용 가능

### <a name="summary"></a>요약

[리소스 갤러리](https://partner.microsoft.com/resources/collection/windows-esu-csp#/)에 Windows 7 Year 2 ESU(Extended Security Updates)가 출시되었습니다.

### <a name="impacted-audience"></a>영향을 받는 대상

CSP(클라우드 솔루션 공급자) 프로그램에 참여하는 모든 파트너

### <a name="details"></a>세부 정보

Windows 7 지원은 2020년 1월 14일에 종료됩니다. 많은 고객이 Windows 10을 순조롭게 배포하고 있지만, Microsoft는 고객마다 업그레이드 프로세스 진행 상황이 다르다는 것을 잘 알고 있습니다. Windows 7 Year 1 ESU의 유효 기간은 2021년 1월 12일에 종료됩니다. 고객이 서서히 내년도 계획을 세우기 시작할 시기입니다.  

2021년에도 Windows 7 디바이스를 계속 사용해야 하는 고객의 경우 고객의 Windows 10 결정을 도와줄 CSP(클라우드 솔루션 공급자) 파트너용 Windows 7 Year 2 ESU가 출시되었습니다. 자세한 내용은 [리소스 갤러리](https://partner.microsoft.com/resources/collection/windows-esu-csp#/)를 참조하세요.

## <a name="partner-center-relationship-removal-feature-update"></a><a name="5"></a>파트너 센터 관계 제거 기능 업데이트

### <a name="categories"></a>범주

- 날짜: 2020-06-29
- 다음을 입력합니다. 기능
- 상태: 지금 이용 가능
- 
### <a name="summary"></a>요약

2020년 6월 29일부터 CSP(클라우드 솔루션 공급자) 간접 공급자 및 간접 재판매인은 파트너 센터를 사용하여 다운스트림 관계를 제거할 수 있으며, 더 이상 다운스트림 관계를 제거하기 위해 파트너 센터 지원 티켓을 제출할 필요가 없습니다. 따라서 파트너는 이 작업을 수행할 때 상당한 시간을 절약할 수 있습니다.

### <a name="impacted-audience"></a>영향을 받는 대상

CSP(클라우드 솔루션 공급자) 간접 공급자 및 간접 재판매인

### <a name="scenarios-supported"></a>지원되는 시나리오:

- 간접 재판매인과 관계를 설정한 간접 공급자는 파트너 센터를 사용하여 해당 관계를 제거할 수 있습니다.
- 최종 고객과 관계를 설정한 간접 재판매인은 파트너 센터를 사용하여 해당 관계를 제거할 수 있습니다.

위 시나리오의 작업은 파트너 센터 관리 에이전트에서 수행해야 합니다.

### <a name="next-steps"></a>다음 단계:

- 자세한 내용은 [리소스 갤러리](https://partner.microsoft.com/resources/collection/partner-center-relationship-removal#)를 방문하세요.
- 작업 준비 갤러리 컬렉션 자료를 검토합니다. 파트너 센터 관계 제거 기능 업데이트
- 적절한 팀에서 프로세스를 검토하고 업데이트할 수 있도록 이 정보를 조직 내에서 공유하세요.

## <a name="billing-frequency-default-changing-from-annual-to-monthly"></a><a name="4"></a>연간에서 월간으로 청구 주기 기본값 변경

### <a name="categories"></a>범주

- 날짜: 2020-06-17
- 다음을 입력합니다. 기능
- 상태: 곧 공개 예정

### <a name="summary"></a>요약

파트너 센터 구매 환경에서 청구 빈도 기본값이 연간에서 월간으로 변경됩니다.

### <a name="impacted-audience"></a>영향을 받는 대상

CSP(클라우드 솔루션 공급자) 직접 공급자 및 간접 공급자

### <a name="details"></a>세부 정보

청구 빈도 설정은 파트너가 구매한 서비스에 대해 청구되는 빈도를 결정합니다. 파트너는 파트너 센터 카탈로그에서 구매할 때 **검토** 단계에서 이 설정을 볼 수 있습니다. 라이선스 기준 서비스에 대해 연간 청구서가 도입된 이후 이 설정은 **매년**으로 기본 설정되었습니다. 파트너 피드백을 기반으로 하는 파트너 센터 팀은 6월 17일부터 **매년**에서 **매월**로 이 기본값을 변경하여 파트너가 기대하는 것과 더 잘 일치시킵니다. 파트너는 검토 페이지에서 트랜잭션을 제출하기 전 또는 [구독 관리 페이지에서 트랜잭션을 수행한 후]( https://docs.microsoft.com/partner-center/billing-annual-monthly#change-billing-frequency) 변경할 수 있습니다. 연간 청구만 지원하는 제품은 계속 매년으로 기본값을 유지합니다. 이 기본값 변경은 파트너 센터 포털 사용자 환경에만 적용되며 파트너 센터 API의 작동 방식에 대한 변경 사항은 없습니다.

이러한 변경으로 파트너가 실수로 연간 청구된 구독을 실수로 구매할 가능성이 줄어들기를 바랍니다.

## <a name="changes-to-microsoft-365-f1"></a><a name="3"></a>Microsoft 365 F1에 대한 변경 사항

### <a name="categories"></a>범주

- 날짜: 2020-06-15
- 다음을 입력합니다. 제품/시장
- 상태: 지금 이용 가능

### <a name="summary"></a>요약

Microsoft 365 F1이 새로운 서비스로 업데이트됩니다. 6월 15일에 이전 F1을 대체하기 위해 새 제품이 출시됩니다.

### <a name="impacted-audience"></a>영향을 받는 대상

CSP(클라우드 솔루션 공급자) 직접 공급자 및 간접 공급자

### <a name="details"></a>세부 정보

Microsoft 365 F1은 2020년 4월 이후에 파트너 센터에서 사용할 수 있습니다. 새로운 서비스에 대한 지원을 추가하여 다음과 같은 향상된 기능을 제공하고 있습니다. Exchange 키오스크 및 MS 검색 기능. 새로운 서비스는 6월 말까지 원래 F1 제공에 따라 기존 구독에 대해 자동으로 프로비저닝됩니다. 기존 F1 제품은 새 구독 구매를 위해 6월 15일 새로운 F1 제품(아래)으로 교체됩니다.

기존 F1 제품:

   |**제품 이름**|**제품 ID**|**소재 ID**|**프로비저닝 ID**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|1ce4a4ba-65f7-49e7-bb28-d92d07d58b86|1PI-00005|M365_F1|
   |Microsoft 365 F1 평가판|dd97e36a-f0df-4a9a-b461-625420656511|1PI-00006|M365_F1|

새 F1 구독을 구매하려는 파트너는 6월 15일부터 아래 제품 ID를 사용해야 합니다. 파트너는 파트너 센터 카탈로그에서 이러한 ID를 볼 수 있습니다. 파트너 센터 API를 활용하는 파트너도 아래 정보를 사용할 수 있습니다.

새 F1 제품:

   |**제품 이름**|**제품 ID**|**소재 ID**|**프로비저닝 ID**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|17b4a1d6-2c65-478e-a046-51cf40d25d76|1PI-00012|M365_F1_COM|
   |Microsoft 365 F1 평가판|2e4e9424-06db-4d35-b2d8-3a1b0ef1dc63|1PI-00013|M365_F1_COMM|

### <a name="call-to-action"></a>조치 사항
현재 Microsoft 365 F1 제품을 사용하는 파트너에는 취할 조치가 없습니다. Exchange 키오스크 및 MS Search의 서비스 플랜은 기존 서비스 플랜에 대해 6월 말까지 백필될 예정입니다.

6월 15일 이후에 Microsoft 365 F1 키를 구매하려는 파트너는 새 F1 제품 정보를 사용하여 올바른 제품을 구매하고 있는지 확인해야 합니다. 새 F1 제품 ID 및 기타 정보는 7월 1일 CSP(클라우드 솔루션 공급자) 가격 목록 업데이트에서 업데이트될 예정입니다.
