## <a name="upcoming-new-microsoft-dynamics-365-offers"></a><a name="3"></a>예정된 새로운 Microsoft Dynamics 365 제품

### <a name="categories"></a>범주

- 날짜: 2020-09-01
- 제품/시장

### <a name="impacted-audience"></a>영향을 받는 대상

Microsoft Dynamics 365를 거래하는 파트너

### <a name="summary"></a>요약

Microsoft Dynamics 365 고객을 위한 후속 제품에는 Microsoft Project Operations 릴리스 및 Business Central 추가 환경이 포함됩니다.

### <a name="details"></a>세부 정보

2020년 10월 1일에 제공되는 새 제품은 다음과 같습니다.

#### <a name="project-operations"></a>프로젝트 작업

2020년 10월 1일부터 Microsoft는 Dynamics 365 Project Operations 제품을 릴리스하고, CSP(클라우드 솔루션 공급자) 프로그램에 대한 Dynamics 365 PSA(Project Service Automation)를 제거합니다. 새 고객은 Dynamics 365 Project Operations를 구매해야 합니다. 

PSA를 사용하는 기존 Dynamics 365 고객의 경우:

- 2020년 10월 1일까지 또는 가장 최근의 계약 또는 갱신이 종료될 때까지(둘 중 먼저 발생하는 날짜까지) Dynamics 365 PSA를 계속 사용합니다.
- Dynamics 365 PSA는 CSP에서 2021년 10월 1일까지 1년간 자동으로 갱신되며, 계약은 2022년 10월 1일까지 계속됩니다.
- Dynamics 365 PSA 라이선스는 2020년 12월 31일까지 현재 라이선스의 일부로 Project Operations를 자동으로 받습니다.
- Dynamics 365 PSA 계약이 종료되면 고객은 사용자 최소 PSA 전환 SKU가 0인 Project Operations로 전환해야 합니다.

제품 ID 세부 정보는 [Dynamics CSP 제품 - 2020년 10월 Excel 문서](https://partner.microsoft.com/resources/detail/microsoft-dynamics-365-new-ids-offers-csp-october-2020-xls)의 Project Operations 탭을 검토하세요.

#### <a name="additional-resources"></a>추가 자료

- PSA를 위한 [전환 리소스](https://partner.microsoft.com/asset/collection/psa-transition-resources#/)
- [Microsoft Dynamics 365 Project Service Automation 소개](https://cloudblogs.microsoft.com/dynamics365/bdm/2020/02/20/introducing-microsoft-dynamics-365-project-operations/) 블로그
- [Microsoft Dynamics 365 Project Service Automation의 기능](https://assetsprod.microsoft.com/mpn/what-is-coming-in-d365-project-service-automation-dyn767pal.pdf) 학습 자료

#### <a name="expansion-of-business-central-offers-into-additional-countries"></a>Business Central 제품을 추가 국가로 확장

2020년 10월 1일부터 Microsoft는 Business Central 클라우드 제품을 브라질, 아일랜드, 리투아니아, 페로 제도 및 그린란드로 확장합니다.

#### <a name="business-central-additional-environments"></a>Business Central 추가 환경

2020년 10월 1일부터 Business Central 클라우드 제품에 대한 새로운 추가 기능 제안을 통해 고객이 추가 환경을 추가할 수 있습니다. 

제품 ID 세부 정보는 [Dynamics CSP 제품 - 2020년 10월 Excel 문서](https://partner.microsoft.com/resources/detail/microsoft-dynamics-365-new-ids-offers-csp-october-2020-xls)의 **추가 환경** 탭을 검토하세요.

### <a name="next-steps"></a>다음 단계

이 업데이트를 조직의 관련 담당자와 공유합니다.

## <a name="office-365-calling-plan-will-be-available-in-new-markets-on-october-1-2020"></a><a name="2"></a>Office 365 통화 플랜은 2020년 10월 1일에 새로운 시장에서 사용 가능

### <a name="categories"></a>범주

- 날짜: 2020-09-01
- 제품/시장

### <a name="impacted-audience"></a>영향을 받는 대상

CSP(클라우드 솔루션 공급자) 파트너 - 직접 공급자, 간접 공급자 및 간접 재판매인 포함

### <a name="summary"></a>요약

Microsoft는 2020년 10월 1일에 새로운 시장(오스트리아, 덴마크, 이탈리아, 포르투갈, 스웨덴 및 스위스)에서 통화 플랜을 사용할 수 있다고 발표하게 되어 기쁘게 생각합니다. 

### <a name="details"></a>세부 정보

Microsoft 365 통화 플랜은 2020년 10월 1일에 새로운 시장(오스트리아, 덴마크, 이탈리아, 포르투갈, 스웨덴 및 스위스)에서 사용할 수 있습니다. 새로운 지리적 가용성은 EA(기업 계약) 또는 EAS(기업 계약 구독), EES(교육 솔루션 등록), 웹 다이렉트 및 CSP를 통해 상용, 전 세계 공공 부문, 비영리 및 정부 커뮤니티 클라우드에서 시작됩니다. 

모든 통화 플랜 기능은 Microsoft의 국내 및 국제 통화 플랜을 사용할 수 있습니다. 여기에는 p/u/m(월별 사용자당) 기준 120, 240 및 1,200분 국내 플랜 및 p/u/m 기준 국내 1,200분과 국제 600분이 적용되는 국제 플랜이 포함됩니다. 

[최신 작업 라이선스 FAQ](https://partner.microsoft.com/resources/collection/partner-center-edu-validation-enhancements)의 통화 플랜 섹션에서 통화 플랜, 과도한 사용량 및 통신 크레딧에 대한 추가 세부 정보를 확인하세요.

통화 플랜은 현재 벨기에, 캐나다, 프랑스, 독일, 아일랜드, 네덜란드, 푸에르토리코(EA 및 EES로만 제한된 채널 사용 가능), 스페인, 영국 및 미국(오스트레일리아 및 일본에서 타사 통화 플랜 사용 가능)에서 사용할 수 있습니다.
