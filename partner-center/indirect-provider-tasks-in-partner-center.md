# <a name="partner-with-indirect-resellers-in-the-cloud-solution-provider-program"></a>클라우드 솔루션 공급자 프로그램의 간접 재판매인과 파트너

**적절한 역할:** 전역 관리자 | 사용자 관리 관리자 | 관리 에이전트 | 판매 에이전트

이러한 복잡하고 빠르게 변화하는 기술 마켓플레이스를 통해 고객은 기술 솔루션이 제공할 수 있는 경쟁 우위를 원합니다. 이러한 기후에서는 파트너가 파트너-파트너 참여에 투자하여 제품을 확장하고 고객이 더 많은 작업을 수행할 수 있도록 하는 것이 합리적입니다.

새로운 대리점이 시장에 진입하고 시장 및 지역 외부 고객에게 솔루션을 제공하는 방법을 찾고 있습니다. 설정된 기술 공급자는 시장 진출 경로가 될 수 있습니다. CSP(클라우드 솔루션 공급자) 프로그램에서 간접 대리점과 협력하여 다른 파트너와 연결하고, 새로운 솔루션을 출시하고, 고객의 클라우드 서비스 채택을 발전시키세요.

간접 공급자-대리점(2계층) 모델에서는 간접 대리점과 협력하여 고객에게 Microsoft 클라우드 솔루션을 제공합니다. Microsoft에서 직접 제품 및 구독을 구매한 다음, 대리점을 통해 고객에게 이러한 솔루션을 제공합니다.

간접 공급자는 파트너 센터 사용하여 다음을 수행합니다.

- 도달 범위를 확장하고 고객 기반을 확장할 수 있는 간접 대리점과 연결

- 간접 대리점과의 파트너 관계를 설정하여 둘 다 일반 고객에게 서비스를 제공할 수 있습니다.

- 특정 고객 및 구독을 관리하도록 간접 대리점 지정

- Microsoft, 고객 및 간접 대리점과의 관계를 모두 한 곳에서 관리

간접 대리점으로 CSP 프로그램에 참여하는 데 필요한 내용에 대한 자세한 내용은 Microsoft 파트너 네트워크 웹 사이트에서 [간접 대리점으로 시작을 참조하세요.](https://partner.microsoft.com/membership/cloud-solution-provider/) [](https://partner.microsoft.com)

## <a name="indirect-provider-tasks-in-the-partner-center"></a>파트너 센터의 간접 공급자 작업

간접 대리점 및 고객과의 파트너 관계를 설정하고 관리하려면 다음 단계를 완료합니다. 2단계와 3단계는 아래에서 자세히 설명합니다.

1. [Microsoft의 공식 간접 공급자 목록](https://partnercenter.microsoft.com/partner/find-a-provider)에서 회사 프로필을 검토 합니다.

   - 회사의 간접 공급자 프로필을 변경하려면 Microsoft 담당자에게 문의하세요.

2. 간접 대리점을 사용 하 여 연결

   - 대리점과의 상호 작용을 하기 전에 파트너 센터에서 파트너와 파트너 관계를 설정 해야 합니다.

3. 고객과의 관계 구축

   - 대리점에 새 고객과 구독 연결

   - 기존 고객과 대리점 관계 설정

이러한 단계를 완료 한 후에는 **파트너 센터** 메뉴의 **간접 대리점** 페이지에서 재판매인 파트너 관계를 관리할 수 있습니다.

> [!IMPORTANT]
> 대리점 작업을 시작 하려면 먼저 CSP 프로그램에서 간접 대리점으로 등록 하 고 파트너 센터 계정이 있어야 합니다. CSP에서 csp를 등록 하는 방법에 대 한 자세한 내용은 [csp에서 간접 공급자와 파트너](https://msdn.microsoft.com/partner-center/indirect-reseller-tasks-in-partner-center.md) 를 가리키도록 합니다.

## <a name="connect-with-indirect-resellers"></a>간접 대리점을 사용 하 여 연결

파트너 센터에서 간접 대리점을 사용 하 여 상호 작용 하려면 파트너와의 파트너 관계를 설정 해야 합니다. 간접 재판매인과 파트너 관계를 유지 하 여 사용자와 재판매인 모두 일반 고객에 게 서비스를 제공할 수 있습니다.

파트너 관계를 설정 하려면 재판매인에 게 파트너 관계 요청을 전자 메일로 보냅니다. 시작 하기 전에 재판매인의 전자 메일 주소가 있어야 합니다.

구독 정보 페이지의 재판매인 (MPN ID) 필드에 사용할 재판매인을 추가 합니다. 파트너 센터에 등록 하는 데 사용한 재판매인의 파트너 위치 계정과 연결 된 MPN ID를 사용 해야 합니다. 이를 통해 재판매인은 적절 한 성과급을 받고 트랜잭션에 참여 하는 활성 Microsoft 파트너 계약을 보유 하 게 됩니다.

1. **파트너 센터** 메뉴에서 **간접 대리점** 을 선택 합니다.

2. **간접 대리점** 페이지에서 **파트너 관계 요청** 을 선택 합니다. 초안 전자 메일 메시지가 표시 됩니다.

3. **전자 메일에서 열기** 를 선택 하 여 기본 전자 메일 응용 프로그램에서 메시지를 열거나 **클립보드로 복사** 를 선택 하 여 메시지를 복사 하 여 전자 메일에 붙여 넣습니다. 완료되면 **완료를** 선택합니다.
    > [!IMPORTANT]  
    >  이메일 메시지의 텍스트를 편집할 수 있지만 파트너 센터 계정에 맞게 개인화되어 있는 링크를 포함해야 합니다.

4. 대리점이 파트너 관계 요청을 수락하면 간접 대리점 목록에 표시되면 다음을 수행할 수 있습니다.

   - 대신 새 고객 추가

   - 고객을 대신하여 주문 제출

   - 구독을 구매한 고객을 대신하여 관리할 수 있도록 구독에 연결

   - 특정 구독에 대한 고객의 지원 연락처로 지정

대리점이 원하는 액세스 수준 및 참여 수준에 따라 아래에 설명된 작업 중 하나 이상을 수행할 수 있습니다.

기존 구독의 경우 간접 대리점과 협력하기로 결정한 경우 간접 대리점을 포함하도록 구독 세부 정보를 업데이트할 수 있습니다.

- 구독 세부 정보 페이지의 **REseller(MPN ID)** 필드에 작업할 대리점을 추가합니다. 이렇게 하면 대리점이 적절한 인센티브가 수신됩니다.

| **이 작업을 수행하려면** | **단계** | **Where** |
|   ------------------   |   -------   |   -----   |
| 대리점이 고객의 계정을 관리하고 구독을 관리할 수 있는 관리자 권한을 갖도록 설정 | 새 고객을 추가할 때 고객 계정에 대리점 연결 | 새 **고객** 페이지에서 |
| 대리점이 Azure 고객의 계정을 관리할 수 있도록 설정|새 고객을 추가할 때 Azure 고객 계정에 대리점 연결|Azure **서비스 포털에서** [사용자 설정을](https://aad.portal.azure.com/#blade/Microsoft_AAD_IAM/UsersManagementMenuBlade/UserSettings) 선택하여 간접 대리점 **테넌트 소유자** 권한을 부여합니다.|
| 인센티브에 대한 대리점 크레딧 제공 | 구독에 대리점 연결(신규 또는 기존) | **새 구독** 페이지에서 |
| 대리점에 대 한 특정 구독에 대 한 고객 지원 위임 | **지정 된 지원 연락처** 목록에서 재판매인을 선택 합니다. | 구독 세부 정보 페이지의 **고객 페이지에서 고객을** 선택한 다음 고객의 **구독** 페이지에서 구독을 선택 합니다. |

## <a name="remove-partnership-with-indirect-resellers"></a>간접 재판매인과 파트너 관계 제거

기존 간접 재판매인과 파트너 관계를 제거 하려면 아래 단계를 수행 합니다. 

1. 대시보드의 파트너 센터 메뉴에서 **간접 대리점** 을 선택 합니다. 

2. **간접 대리점** 페이지에서 **파트너 관계 제거** 를 선택 합니다. 확인을 요청 하는 메시지가 표시 됩니다. 계속 하려면 **예** 를 선택 합니다.

:::image type="content" source="images/ip-ir-relationship.png" alt-text="파트너 관계 제거":::

>[!IMPORTANT]
>간접 재판매인에 연결 된 고객은 제거 되지 않습니다. 부여 된 위임 된 관리자 권한은 변경 되지 않고 그대로 유지 됩니다.
>파트너와의 관계를 중단 하기로 결정 하는 것은 적법 한 비즈니스 목적이 나 고객 또는 파트너와의 계약에 따라 적절 한 결정을 내리는 데 동의 하 고, 타사 클레임에 대 한 Microsoft 무해 한 행위와 반대 되는 것에 동의 하는 것에 동의 하는 것입니다.
>계약에 필요한 경우 고객 또는 파트너에게 계약 종료의 법적 고지 사항을 제공해야 합니다. 이 도구는 고객 또는 파트너에게 알림을 생성하지 않습니다.

3. 파트너 관계를 제거한 간접 재판매인은 목록에 표시 되지 않습니다.
액세스 수준과 참여 수준에 따라이 변경 내용에 대 한 최종 고객 뿐만 아니라 간접 재판매인과 통신 하는 것이 좋습니다.


## <a name="connect-with-customers"></a>고객과의 관계 구축

새 고객을 받을 때 파트너 센터에서 해당 고객에 대 한 계정을 만들어야 합니다. 새 고객 계정을 추가 하는 경우 계정을 관리 하 고 고객의 구독과 서비스를 대신 관리 하도록 특정 대리점을 지정할 수 있습니다.

대리점을 새 고객 계정과 연결 하는 경우 계정 및 구독 관리를 재판매인에 게 위임 합니다. 즉, 재판매인은 고객의 구독 및 서비스에 대 한 관리자 액세스 권한을 보유 하 고 고객을 대신 하 여 관리할 수 있습니다.

### <a name="add-new-customers-and-associate-them-with-indirect-resellers"></a>새 고객을 추가 하 고 간접 대리점에 연결

1. **파트너 센터** 메뉴에서 **간접 대리점을** 선택합니다.

2. 새 고객을 할당하려는 간접 대리점을 찾은 다음, **새 고객 추가를** 선택합니다. 고객 세부 정보 양식에서 간접 대리점이 이미 선택되어 있습니다. 고객의 세부 정보를 입력한 후 **다음: 구독을 선택합니다.**

3. 새 **구독** 페이지에서 고객이 구매할 구독을 선택한 다음, 고객에게 필요한 라이선스 수를 입력합니다.

4. 완료되면 **다음: 검토** 를 클릭합니다.

5. 주문이 올바르면 **제출을** 선택합니다. 주문이 올바르지 않으면 **취소를** 선택하여 전체 작업을 취소하고 다시 시작합니다.

6. 주문을 제출하면 **확인** 페이지가 나타납니다. **설치 정보** 아래에 관리 사용자 계정 이름 및 임시 암호가 표시됩니다. 새 고객에게 보낼 수 있도록 사용자 계정 이름 및 암호 정보를 복사합니다.

    > [!IMPORTANT]  
    > 이 페이지를 닫은 후에는 이 정보에 다시 액세스할 수 없도록 사용자 계정 이름 및 암호를 기록해 두십시오.

7. **완료** 를 선택합니다. 간접 대리점과 연결된 고객 목록에 새 고객이 표시됩니다. 이제 대리점은 고객의 계정 정보 및 구독에 대한 관리자 액세스 권한을 가지며 고객을 대신하여 구독을 관리할 수 있습니다.

### <a name="add-new-subscriptions-to-customer-accounts"></a>고객 계정에 새 구독 추가

고객이 새 구독을 구입하거나, 기존 구독을 업그레이드하거나, 추가 라이선스를 구입하려는 경우 Microsoft 365 포털을 통해 고객의 계정에서 이러한 변경을 합니다. Azure Portal 이 작업을 수행할 수 없습니다.

1. **파트너 센터** 메뉴에서 **고객을** 선택합니다.

2. 계정을 변경하려는 고객을 찾습니다. 간접 대리점별로 목록을 필터링하여 선택 범위를 좁힐 수 있습니다. 아래쪽 화살표를 선택한 다음 **구독 추가** 를 선택하여 고객의 레코드를 확장합니다.

3. **새 구독** 페이지에서 구독을 선택 하 고 고객이 구매할 라이선스의 수를 입력 합니다.

4. 구독과 구독과 연결 하려면 목록에서 재판매인의 이름을 선택 하 고 **제출** 을 선택 합니다. 재판매인을 구독과 연결 하면 해당 하는 경우에는 성과급을 얻을 수 있습니다.

5. 새 구독은 고객의 구독 목록에 표시 됩니다.

6. 재판매인이 새 구독에 대 한 고객 지원 요청을 처리 하도록 하려면 구독을 선택한 다음 **지정 된 지원 담당자** 목록에서 재판매인을 선택 합니다. 

7. 완료 되 면 **제출** 을 선택 합니다.
