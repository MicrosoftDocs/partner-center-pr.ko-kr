# <a name="value-added-tax-vat-information"></a>VAT(부가가치세) 정보

**적절한 역할:** 계정 관리자


파트너 센터 등록 프로세스 중에 VAT ID 번호를 제공해야 하는 경우 시작하는 데 필요한 몇 가지 정보는 다음과 같습니다.

## <a name="understanding-vat-numbers"></a>VAT 번호 이해

VAT(부가가치세) 번호는 유럽 연합에서 국가 또는 지역에 사용되는 식별자입니다. 자세한 내용은 유럽 연합 공식 VIES 사이트 를 [참조하세요.](http://ec.europa.eu/taxation_customs/vies/vieshome.do)

## <a name="location-specific-vat-information"></a>위치별 VAT 정보

다음 세부 정보는 아래에 나열된 국가 또는 지역에 고유합니다.

### <a name="european-union"></a>유럽 연합

유럽 연합 구성원 상태에서 파트너는 역방향 요금 및 적절한 청구를 위해 유효한 VAT ID 번호를 제공해야 합니다. VAT 번호는 유효한 것으로 허용되어야 합니다.

Microsoft는 크레딧 및 다시 청구를 지원하지 않습니다. 구독을 판매하기 전에 [파트너 센터 VAT ID 번호를 등록해야](organization-tax-info.md) 합니다.

### <a name="south-africa"></a>남아프리카 공화국

Microsoft는 이제 남아프리카 공화국에서 파트너가 구매한 디지털 콘텐츠 및 서비스에 표준 VAT를 적용합니다. 청구서에서 VAT를 제외하고 직접 관리하려면 [Microsoft에 VAT ID 번호를 전송하여](organization-tax-info.md)파트너 청구 프로필을 업데이트해야 합니다.

### <a name="turkey"></a>터키

Microsoft는 이제 디지털 콘텐츠 및 파트너가 구매한 서비스에 표준 VAT를 적용합니다. 청구서에서 VAT를 제외하고 직접 관리하려면 [Microsoft에 VAT ID 번호를 전송하여](organization-tax-info.md)파트너 청구 프로필을 업데이트해야 합니다.
